## 📄 Descripción de la práctica
La práctica se centra en la maquetación de una página web personal utilizando estándares modernos. El diseño cuenta con una estructura de **sidebar (barra lateral)** para la información de contacto y un **área principal** para el contenido detallado.

### Características destacadas:
* **Diseño Responsivo:** Adaptación completa a dispositivos móviles mediante el uso de *Media Queries*.
* **Componentes Visuales:** Tarjetas de información interactivas con efectos de resalte (*hover*).
* **Barras de Progreso:** Representación visual de habilidades técnicas (Java, HTML, CSS, MySQL).
* **Interactividad:** Implementación de efectos dinámicos para mostrar imágenes de proyectos al pasar el cursor.

---

## 🛠️ Tecnologías utilizadas
Para el desarrollo de este proyecto se han empleado las siguientes tecnologías:

* **HTML5:** Estructuración semántica del contenido.
* **CSS3:** * **Variables CSS (`:root`):** Para una gestión centralizada de colores.
    * **Flexbox:** Para la alineación de la estructura global y el footer.
    * **CSS Grid:** Para la organización de las tarjetas de formación y habilidades.
    * **Transiciones y Transformaciones:** Para mejorar la experiencia de usuario (UX).
* **Git/GitHub:** Para el control de versiones y el despliegue del proyecto.

---

## 📁 Estructura del Proyecto
* `index.html`: Documento principal con la estructura de la web.
* `CSS/style.css`: Hoja de estilos con variables, diseño de rejillas y media queries.
* `img/`: Directorio con imágenes de perfil, proyectos y patrones de fondo.
